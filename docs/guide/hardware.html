硬件信息查询 ​

通过 Windows Management Instrumentation (WMI) 实时读取系统硬件数据，无需安装任何驱动或第三方组件，信息准确可靠。

支持查询的硬件信息 ​

类别查询内容WMI 类
处理器型号、核心数、线程数、主频、缓存Win32_Processor
内存总容量、可用容量、频率、插槽信息Win32_PhysicalMemory
显卡型号、显存、驱动版本Win32_VideoController
硬盘型号、容量、接口类型、分区信息Win32_DiskDrive
显示器分辨率、刷新率、制造商Win32_DesktopMonitor
网卡型号、MAC 地址、IP 地址Win32_NetworkAdapter
声卡型号、制造商Win32_SoundDevice
主板制造商、型号、BIOS 版本Win32_BaseBoard

特性 ​

  • 实时读取 — 每次打开页面都获取最新数据，不缓存过期信息
  • 后台查询 — WMI 查询在后台线程执行，不阻塞 UI
  • 分类展示 — 按硬件类别分组，信息一目了然
  • 零依赖 — 纯 WMI 查询，无需安装额外驱动或工具
